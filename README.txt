JoshCreativez Visuals — Static Website Template
================================================

Pages:
- index.html (home + contact)
- weddinggalleries.html
- engagementgalleries.html
- details.html (experience + FAQ)
- video.html

How to use:
1) Replace placeholder blocks (the gray "thumb" areas) with your real images.
2) Update brand text, location, and social links in the footer and home card.
3) Connect the contact form to a backend:
   - Formspree (easiest)
   - Netlify Forms
   - Or your own API endpoint

Run locally:
- Just open index.html in your browser
- Or serve with a local server (recommended):
  python -m http.server 8000
  then open http://localhost:8000
